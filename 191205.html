My Note

macOS 系统部署 GitBook 并导出 PDF 的配置步骤

Part 1：安装 node.js

官网下载，直接安装

注意：不建议安装太高版本的 node.js，兼容性有些问题

下载地址：https://nodejs.org/dist/latest-v6.x/node-v6.17.1.pkg

安装完成，输入 node -v 如果显示 node.js 版本代表安装成功

bogon:~ comesoon$ node -v
v6.17.1

Part 2：安装 GitBook

输入命令：sudo npm install gitbook-cli -g

安装完成，输入 gitbook -V 如果显示 GitBook 版本代表安装成功

注意：V 为大写字母

bogon:~ comesoon$ gitbook -V
CLI version: 2.3.2
GitBook version: 3.2.3

Part 3：安装 GitBook 编辑器

安装编辑器主要是为了方便编写 GitBook 书内容

下载地址：https://legacy.gitbook.com/editor/osx

注意：如果地址失效，需要查找第三方地址下载

GitBook 常用命令：

  • 生成静态页面：gitbook build
  • 使用本地预览：gitbook serve

Part 4：安装 calibre 客户端

GitBook 导出 PDF 格式主要依赖 calibre 的组件

下载地址：https://download.calibre-ebook.com/3.48.0/calibre-3.48.0.dmg

注意：要注意 macOS 系统版本，下载对应版本的 calibre

安装完成后，输入命令：sudo ln -s /Applications/calibre.app/Contents/MacOS/ebook-convert /usr/local/bin 以便能够调用 calibre 的组件

生成 PDF 文件的方法：

  • 进入 GitBook 书源文件目录下
  • 输入命令：gitbook pdf . PDF_name.pdf

Part 5：恭喜完成！
